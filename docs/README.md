# BRIDGE Documentation Map

このディレクトリは、BRIDGE の現行文書を役割ごとに整理した入口です。

## Canonical Documents

- `ARCHITECTURE_RULE.md`: リポジトリ全体の最上位アーキテクチャ規範
- `WORD_DEFINITION.md`: 用語の正本

## Specifications

- `algorithms/`: アルゴリズム仕様
- `architecture/`: 公開契約、構成仕様、データ仕様
- `components/`: コンポーネント規則の索引
- `sdk/`: SDK 契約と利用仕様

## Lifecycle Documents

- `migration/`: 移行方針、移行ガイド、移行完了基準
- `plans/`: 実装計画、補正計画、開発タスク
- `operations/`: 運用手順、計測是正、実行ガイド

## Reports

- `reports/audits/`: 監査、完了判定、チェックリスト
- `reports/benchmarks/`: ベンチマーク結果、性能評価
- `reports/implementation/`: 実装報告、改修サマリ
- `reports/migration/`: 移植評価、互換性・移行進捗
- `reports/validation/`: 検証、意味論・契約整合の報告
- `reports/data/`: レポート付随データ
- `reports/timing-regression/`: timing regression の再現用成果物

## Repository Metadata

- `repository/CHANGELOG.md`: 変更履歴
- `repository/STRUCTURE.md`: リポジトリ配置規則
- `repository/FILE_MANIFEST.txt`: 文書と主要資産の一覧
