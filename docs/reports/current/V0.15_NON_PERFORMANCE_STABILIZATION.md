# BRIDGE v0.15 非性能領域安定化

> Status: Current
> Scope: repository hygiene, documentation authority, release integrity

## 方針

性能特性、探索速度、Allocation、Work効率には変更を加えない。今回の変更は、現行正本の明確化、死んだ参照の除去、生成物混入防止、保守用検査の正常化に限定する。

## 実施内容

- 現行仕様と履歴資料を分離した。
- 過去の監査・実装・移行・ベンチマーク文書を`docs/reports/archive/`へ集約した。
- 存在しない`others/legacy`を参照する互換試験を廃止し、現行用語契約検査へ置換した。
- 製品入口ではない比較Runnerを削除した。
- 実行時生成Artifactをソースツリーから削除し、`.gitignore`へ追加した。
- 手管理されていた巨大な`FILE_MANIFEST.txt`を廃止した。

## 非変更保証

- ANCHORおよびBRIDGEの探索実装は変更しない。
- Route、Benchmark、Artifact、SDK、ServerのSchema v1契約は変更しない。
- 性能基準値や退行閾値は追加しない。
