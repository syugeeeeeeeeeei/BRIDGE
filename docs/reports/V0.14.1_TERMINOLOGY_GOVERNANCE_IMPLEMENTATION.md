# v0.14.1 用語管理実装記録

## 実装内容

- BRIDGE固有の定義・意味・用法を持つ語を用語集へ同時記載する規則を追加した。
- 一般語をBRIDGE固有の意味へ限定して使用する場合も用語集対象とした。
- Scenario、Run、Raw Run、Query、Graph Instance、Execution Target、Execution Path、Observation Mode、Time Breakdown、System Metrics等を定義した。
- Stable DigestとEffective Configuration Digest、Solver TimeとEnd-to-End Time等の混同しやすい語を区別した。
- `docs/WORD_DEFINITION.md`と`docs/ARCHITECTURE_RULE.md`を規範上の正本位置へ移した。
- `tests/compatibility/verify_glossary.py`を追加し、正本の存在、必須用語、更新義務を検査するようにした。

## 運用規則

公開schema、CLI、Scenario、result、trace、component、algorithm機構、研究指標へ独自語を追加または変更する変更は、用語集の更新を伴わなければ完了とみなさない。
